1. Tạo file .env
2. Tạo database
3. cài gói npm:
npm install
4. Cai goi composer:
composer install
5. Tao database
php artisan make:migration create_users_table --create=users
6. Them truong vao bang da co
php artisan make:migration add_votes_to_users_table --table=users

II Quy trinh tạo API
1. Tạo request
php artisan make:request BiometricRequest

Tạo Models
php artisan make:model Biometric	
2. Tạo UserInterface.php, place the file in your app/Interfaces folder. 

3. Tạo Controller API
php artisan make:Controller Api/BiometricController --api
4. Tạo UserRepostitory.php, place the file in your app/Repositories folder.

5 Tao API Resources trả về giá trị truy vấn liên kết các bảng
php artisan make:resource Biometric/BiometricResource
php artisan make:resource Biometric/BiometricCollection
